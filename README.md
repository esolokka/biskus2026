# BiSKuS 2026 – ZfA DSD-Lehrerseminar Bonn

Interaktive 3D-Weltkarten-Visualisierung der teilnehmenden DSD-Ortslehrkräfte (OLK) für das Bildungsseminar für Kulturarbeit und Sprache (BiSKuS) 2026 in Bonn, Deutschland.

## Features
- **3D-Globus**: Three.js & Globe.gl Visualisierung mit Ländergrenzen.
- **Teilnehmerliste**: 26 internationale Lehrkräfte aus aller Welt mit Distanzangaben.
- **Interaktiver Fokus**: Kamera-Rotation und Zoom bei Auswahl eines Teilnehmers auf dem Globus oder in der Liste.
- **Seminar-Protokoll**: Live-Simulationsprotokoll der Seminaraktivitäten.
